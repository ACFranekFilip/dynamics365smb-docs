# Fixed Asset Turnover
The fixed asset turnover feature enables you to:  
  
-   Control the turnover of fixed assets or certain groups of fixed assets.  
  
-   View useful information about fixed assets in one place.  
  
-   Compare the posting of one fixed asset in different fixed asset depreciation books.  
  
-   Post fixed assets financial results monitoring.  
  
 The following reports and windows are part of fixed assets:  
  
-   **\($ R\_12466 FA Turnover $\)** report  
  
-   **\($ R\_14970 Comparing Depr. Book Entries $\)** report  
  
-   **\($ N\_12497 FA Sheet $\)** window  
  
-   **\($ N\_12499 FA G\/L Turnover $\)** window  
  
-   **\($ R\_12464 Fixed Asset G\/L Turnover $\)** report  
  
## Fixed Asset Turnover Report  
 The **\($ R\_12466 FA Turnover $\)** report shows the fixed asset turnover. Use the report to view information such as the fixed asset name, quantity, status, depreciation dates, and amounts. The report can be used as documentation for the correction of quantities or for auditing.  
  
 For more information, see [\($ R\_12466 FA Turnover $\)](../../LocalFunctionalityForMicrosoftDynamicsNav2016/Russia/-$-r_12466-fa-turnover-$-.md).  
  
## Comparing Depreciation Book Entries Report  
 The **\($ R\_14970 Comparing Depr. Book Entries $\)** report shows the amounts of fixed asset operations of the main types in two selected depreciation books for all selected fixed assets. You can view the amounts of the acquisition cost, write\-down, appreciation, and depreciation. The report can be used for auditing the posting of fixed asset operations.  
  
 For more information, see [\($ R\_14970 Comparing Depr. Book Entries $\)](../../LocalFunctionalityForMicrosoftDynamicsNav2016/Russia/-$-r_14970-comparing-depr.-book-entries-$-.md).  
  
## Fixed Asset Sheet Window  
 The **\($ N\_12497 FA Sheet $\)** window shows information similar to the **\($ R\_12466 FA Turnover $\)** report, but in a window. It has some differences in the options set, in extra filtering functionality, and in fields in the layout. The window shows fixed asset turnover. It runs on the basis of fixed asset depreciation books.  
  
 For more information, see [\($ N\_12497 FA Sheet $\)](../../LocalFunctionalityForMicrosoftDynamicsNav2016/Russia/-$-n_12497-fa-sheet-$-.md).  
  
## Fixed Asset General Ledger Turnover Window  
 The **\($ N\_12499 FA G\/L Turnover $\)** window shows financial turnover as a result of fixed asset posting. General ledger entries are the basis for amounts shown in the window. You can define what is shown in the window by setting filters.  
  
 For more information, see [\($ N\_12499 FA G\-L Turnover $\)](../../LocalFunctionalityForMicrosoftDynamicsNav2016/Russia/-$-n_12499-fa-g-l-turnover-$-.md).  
  
## Fixed Asset General Ledger Turnover Report  
 The **\($ R\_12464 Fixed Asset G\/L Turnover $\)** report is similar to the **\($ N\_12499 FA G\/L Turnover $\)** window, but it is in a printing layout, and has differences in the options and in the information and fields that can be shown in the layout. You can define what is included in the report by setting filters.  
  
 For more information, see [\($ R\_12464 Fixed Asset G\-L Turnover $\)](../../LocalFunctionalityForMicrosoftDynamicsNav2016/Russia/-$-r_12464-fixed-asset-g-l-turnover-$-.md).
